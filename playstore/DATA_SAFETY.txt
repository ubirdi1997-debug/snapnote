==========================================
DATA SECURITY
==========================================

HOW DO YOU SECURE USER DATA?
Answer: All data is stored locally on the device using local database storage.
No data is transmitted over the internet.

==========================================
DATA DELETION
==========================================

CAN USERS REQUEST DATA DELETION?
Answer: Users can delete all data by:
1. Deleting notes within the app
2. Clearing app data in device settings
3. Uninstalling the app

All data is stored locally and can be completely removed by the user.

==========================================
PERMISSIONS EXPLANATION
==========================================

MICROPHONE PERMISSION:
- Purpose: Voice note recording and speech-to-text conversion
- Data Handling: Audio is processed locally on device using offline speech recognition
- Data Transmission: NONE - All processing is offline
- Data Storage: Only transcribed text is stored locally in notes

CAMERA PERMISSION:
- Purpose: Image capture for text extraction via OCR
- Data Handling: Images processed locally using offline ML Kit OCR
- Data Transmission: NONE - All processing is offline
- Data Storage: Images are temporarily stored for processing, then immediately deleted. Only extracted text is saved in notes.

==========================================
THIRD-PARTY SERVICES
==========================================

GOOGLE ML KIT TEXT RECOGNITION:
- Usage: Offline OCR for text extraction from images
- Data Transmission: NONE - All processing occurs locally on device
- Data Collection: NONE - No data is sent to Google

SPEECH-TO-TEXT:
- Usage: Offline speech recognition for voice notes
- Data Transmission: NONE - All processing occurs locally on device
- Data Collection: NONE - No audio data is transmitted
